# triangle.css
Easy to use Responsive CSS-Triangles ect

## How to install?
```
<link rel="stylesheet" href="triangle.css">
```

## How does it work?

### Classes
The base-construct of a Triangle looks like this:
```
<div class="tri-fluid">
    <div class="tri-left-to-right b-dark"> </div>
</div>
```
